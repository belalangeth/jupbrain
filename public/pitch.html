JUPBRAIN

The Ultimate Solana Native Terminal

Jupiter Developer Bounty Submission

github.com/belalangeth/jupbrain

The Problem

  • Solana's DeFi ecosystem is fragmented across isolated UI widgets.
  • Third-party iframe widgets are great for simple integrations, but heavily restrict UI/UX customization.
  • Developers struggle to combine deep liquidity routing and prediction betting into a single, cohesive, brandable experience.

Our Solution

We bypassed the widgets and went straight to the metal.

  • A custom, retro-futuristic React dashboard built natively.
  • Combines Swaps, Prediction Markets, and Token Data seamlessly.
  • Full control over the UI, resulting in a cohesive terminal aesthetic.

The Tech Stack

Core APIs

  • Jupiter Swap V2 (/order & /execute)
  • Jupiter Prediction V1
  • Jupiter Tokens V2

Frontend

  • Next.js & React
  • @solana/web3.js
  • Vanilla Custom CSS

Developer Experience

Deeply integrating the API revealed critical edge cases:

  • Micro-USD Quirks: Prediction API heavily utilizes undocumented Micro-USD values for odds & volume.
  • Swap V2 Payloads: Discovered silent 200 OK responses with empty transactions due to "Insufficient Funds", bypassing standard REST error handling.
  • Outcome: We wrote a highly detailed DX Report (DX-REPORT.md) to help the Jupiter team improve API standardizations.
